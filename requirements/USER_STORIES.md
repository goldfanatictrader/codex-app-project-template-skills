# User Stories

## Format
```
As a [user type]
I want to [action]
So that [benefit]

Priority: P0/P1/P2/P3
Story Points: 1/2/3/5/8/13
Sprint: [Number]
```

---

## P0 - Must Have (MVP)

### US-001: [User Authentication]
```
As a new user
I want to create an account
So that I can access the platform

Priority: P0
Story Points: 5
Sprint: 1

Acceptance Criteria:
- [ ] Given I'm on the registration page when I enter valid email/password then account is created
- [ ] Given I'm on the registration page when I enter invalid email then error is shown
- [ ] Given I'm on the registration page when password < 8 chars then error is shown
- [ ] Given I'm a registered user when I login with correct credentials then I'm redirected to dashboard
- [ ] Given I'm logged in when I click logout then session is terminated
```
